# قيد استراحتنا — موقع التطبيق الرسمي

موقع ويب احترافي لتطبيق "قيد استراحتنا" — التطبيق الأمثل لحساب نقاط البلوت والهند.

## 📱 عن التطبيق

"قيد استراحتنا" هو تطبيق متخصص لحساب نقاط ألعاب البلوت والهند، مصمم لتسهيل تتبع النقاط وإدارة جلسات اللعب مع الأصدقاء.

### ✨ المميزات الرئيسية

- **قيد البلوت** — احسب نقاط الجولات بدقة وتابع موزع الورق
- **قيد الهند** — دعم كامل للعب الفردي والفرق مع حساب الخالص تلقائياً
- **عجلة العقوبات** — أضف مرحاً على جلستك بالعجلة التفاعلية
- **وضع ليلي** — تصميم مريح للعيون في الظلام
- **سجل اللعبات** — تاريخ كامل محفوظ تلقائياً
- **استراحتنا+** — اشتراك بدون إعلانات

الموقع جاهز للنشر على GitHub Pages أو أي خدمة استضافة ساكنة.

### المتطلبات
- لا توجد متطلبات تشغيلية (موقع ساكن)
- متوافق مع جميع المتصفحات الحديثة
- متجاوب مع الأجهزة المحمولة

### خطوات النشر على GitHub Pages

1. ارفع المشروع إلى GitHub
2. من إعدادات المستودع، فعّل GitHub Pages
3. اختر المصدر `main` branch
4. سيكون الموقع متاحاً على: `https://username.github.io/repository-name`
